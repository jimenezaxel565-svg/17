gap > margin

Deja de usar margin para separar elementos

La propiedad gap de Flexbox es lo mejor que le pasó al diseño web moderno.

.item { margin-right: 1rem; }
.item:last-child { margin-right: 0; }
Selectores feos, wrap roto
Problema

Margin en Flexbox es tedioso

/* ❌ Solución antigua */
.flex-container {
  display: flex;
  flex-wrap: wrap;
}
.flex-item {
  margin-right: 16px;
  margin-bottom: 16px;
}
.flex-item:last-child {
  margin-right: 0;
}
/* ¿Y si hay wrap? El margen inferior sobra en la última fila */

Necesitas selectores extra, el último elemento sobra, y en wrap el margen inferior también molesta.

.flex { gap: 16px; }
Solo una línea, sin selectores
Solución

gap en Flexbox (y Grid)

/* ✅ Moderno y limpio */
.flex-container {
  display: flex;
  flex-wrap: wrap;
  gap: 16px;      /* separa hijos horizontal y vertical */
  row-gap: 20px;  /* opcional: separación específica */
  column-gap: 12px;
}

/* Sin márgenes extra, sin pseudoclases, sin preocupaciones */

gap añade espacio entre los hijos, no alrededor. Funciona perfecto con flex-wrap: wrap.

/* Ejemplo real */
.nav {

  gap: 1.5rem;
}
.grid {
  display: grid;
  gap: 1rem;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
